## Konane Game Engine (Python)

This project is a Python implementation of the board game Konane (Hawaiian checkers). It simulates gameplay by generating the board, validating moves, and updating the game state.

### Features
- Generates an NxN board with alternating pieces  
- Validates moves based on Konane rules  
- Includes human, random, and AI players  
- Uses a simple AI that selects moves to capture the most pieces  

### Technologies
- Python
